Determines how the content is distributed across the main axis

4.1 Justify content - flex-start (Default)

4.2 Justify content - flex-end

4.3 Justify content - center

4.5 Justify content - space-around

5. Flex Wrap

Determines whether elements are going to wrap along the main axis onto a new line

5.1 Flex Wrap - wrap

5.2 Flex Wrap - wrap-reverse

Note: Now, cross axis goes from bottom to top

Note: Flex wrap can explicity set to no-wrap
